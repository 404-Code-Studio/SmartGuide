Grundlagen

Nachrichten

Verschicken und empfangen.
E-Mails, SMS oder WhatsApp-Nachrichten
Tippen zum Durchlesen

Nachrichten verschicken - so geht's:
Es gibt verschiedene Apps für Nachrichten (WhatsApp, SMS, E-Mail). Die grundlegenden Schritte sind bei allen ähnlich.

  1. App finden: Wischen Sie mit dem Finger von oben nach unten über den Bildschirm. Es öffnet sich die Suche.
  2. App öffnen: Tippen Sie "WhatsApp" oder "Nachrichten" in die Suchleiste ein. Berühren Sie dann die entsprechende App, wenn sie erscheint.
  3. Kontakt auswählen: Suchen Sie den gewünschten Kontakt und tippen Sie auf die Unterhaltung. Tippen Sie dann auf das Textfeld ganz unten.
  4. Nachricht schreiben: Tippen Sie Ihre Nachricht ein und drücken Sie auf den Pfeil rechts, um sie zu versenden.
Tipp: Wenn Ihr Kontakt antwortet, wissen Sie, dass die Nachricht angekommen ist.

Anrufe

Anrufen und Anrufe annehmen
Tippen zum Durchlesen

Anrufe tätigen:

  1. Öffnen Sie die "Telefon"-App (grünes Symbol mit Telefonhörer)
  2. Wählen Sie eine Option:
    • Kontakt anrufen: Tippen Sie auf "Kontakte" und wählen Sie eine Person aus
    • Nummer eingeben: Tippen Sie auf "Ziffernblock" und geben Sie die Telefonnummer ein
  3. Tippen Sie auf den grünen Anruf-Button

Anrufe annehmen:

  1. Wenn das Telefon klingelt, erscheint ein Bildschirm mit dem Namen oder der Nummer
  2. Schieben Sie den grünen Button nach rechts, um anzunehmen
  3. Schieben Sie den roten Button nach links, um abzulehnen
Tipp: Während eines Anrufs können Sie auf "Stumm" tippen, um Ihr Mikrofon auszuschalten, oder auf "Lautsprecher" für Freisprechen.

Apps

Finden und öffnen
Tippen zum Durchlesen

Apps finden und öffnen:

Methode 1: Über die Suche

  1. Wischen Sie mit dem Finger von oben nach unten über den Bildschirm
  2. Geben Sie den Namen der App in die Suchleiste ein (z.B. "Wetter", "Kamera")
  3. Tippen Sie auf die App, um sie zu öffnen

Methode 2: Auf dem Homebildschirm

  1. Schauen Sie auf Ihren Homebildschirm (Hauptbildschirm)
  2. Wischen Sie nach links oder rechts, um weitere App-Seiten zu sehen
  3. Tippen Sie auf die gewünschte App
Tipp: Die Suchfunktion ist oft der schnellste Weg, eine App zu finden, besonders wenn Sie viele Apps haben.

WLAN und mobile Daten

Was ist der Unterschied?
Tippen zum Durchlesen

Der Unterschied zwischen WLAN und mobilen Daten:

WLAN

  • Was ist das? Ein drahtloses Netzwerk zu Hause, im Büro oder an öffentlichen Orten
  • Vorteil: Meist unbegrenzte Internetnutzung ohne zusätzliche Kosten
  • Nachteil: Nur in der Nähe des WLAN-Routers verfügbar

Mobile Daten

  • Was ist das? Internet über das Mobilfunknetz (überall verfügbar)
  • Vorteil: Funktioniert unterwegs, auch ohne WLAN
  • Nachteil: Oft begrenzte Datenmenge pro Monat
Wichtiger Tipp: Nutzen Sie zu Hause immer WLAN, um Ihre mobilen Daten zu schonen. Mobile Daten sollten nur unterwegs verwendet werden, wenn kein WLAN verfügbar ist.

Kontakte

Hinzufügen und verwalten
Tippen zum Durchlesen

Kontakt hinzufügen

  1. Öffnen Sie die "Telefon"-App
  2. Tippen Sie unten auf "Kontakte"
  3. Tippen Sie auf das blaue Plus-Zeichen (+) oben rechts
  4. Geben Sie den Namen und die Telefonnummer ein
  5. Tippen Sie oben rechts auf "Sichern"

Kontakt bearbeiten

  1. Öffnen Sie die "Telefon"-App und gehen zu "Kontakte"
  2. Suchen Sie den gewünschten Kontakt
  3. Tippen Sie auf den Namen
  4. Tippen Sie oben rechts auf "Bearbeiten"
  5. Nehmen Sie Ihre Änderungen vor und tippen auf "Fertig"

Kontakt blockieren

  1. Öffnen Sie den Kontakt, den Sie blockieren möchten
  2. Scrollen Sie nach unten
  3. Tippen Sie auf "Kontakt blockieren" (rot)
  4. Bestätigen Sie mit "Kontakt blockieren"
Tipp: Blockierte Kontakte können Sie nicht mehr anrufen oder Ihnen Nachrichten schicken. Sie können die Blockierung in den Einstellungen unter "Telefon" → "Blockiert" wieder aufheben.

Passwort

Ändern und verwalten
Tippen zum Durchlesen

Bildschirmsperre-Code ändern:

  1. Entsperren Sie Ihr iPhone
  2. Öffnen Sie die "Einstellungen"-App
  3. Tippen Sie auf "Face ID & Code" (oder "Touch ID & Code")
  4. Geben Sie Ihren aktuellen Code ein
  5. Scrollen Sie nach unten und tippen auf "Code ändern"
  6. Geben Sie Ihren alten Code ein
  7. Geben Sie Ihren neuen Code ein
  8. Geben Sie den neuen Code zur Bestätigung noch einmal ein
Sicherheitstipp: Verwenden Sie einen Code, den Sie sich gut merken können, aber den andere nicht erraten können. Vermeiden Sie einfache Zahlenfolgen wie "1234" oder Ihr Geburtsjahr.

WLAN-Verbindung erstellen

Mit dem Internet zu Hause verbinden
Tippen zum Durchlesen

WLAN einrichten - Schritt für Schritt:

  1. Entsperren Sie Ihr iPhone
  2. Öffnen Sie die "Einstellungen"-App
  3. Tippen Sie auf "WLAN" (ganz oben)
  4. Stellen Sie sicher, dass WLAN aktiviert ist (grüner Schalter)
  5. Suchen Sie in der Liste den Namen Ihres Heimnetzwerks
  6. Tippen Sie auf den Namen Ihres Netzwerks
  7. Geben Sie das WLAN-Passwort ein
  8. Tippen Sie auf "Verbinden"
Wo finde ich mein WLAN-Passwort? Das Passwort steht meist auf einem Aufkleber auf Ihrem Router oder wurde Ihnen von Ihrem Internetanbieter mitgeteilt. Oft steht dort "WLAN-Schlüssel" oder "WiFi-Password".

Backups erstellen

Ihre Daten sichern
Tippen zum Durchlesen

Was ist ein Backup?
Ein Backup ist eine Sicherheitskopie all Ihrer wichtigen Daten (Fotos, Kontakte, Nachrichten). Falls Ihr Handy kaputt geht oder verloren geht, können Sie alles wiederherstellen.

Backup erstellen:

  1. Öffnen Sie die "Einstellungen"-App
  2. Tippen Sie ganz oben auf Ihren Namen
  3. Tippen Sie auf "iCloud"
  4. Tippen Sie auf "iCloud-Backup"
  5. Stellen Sie sicher, dass "iCloud-Backup" aktiviert ist (grüner Schalter)
  6. Tippen Sie auf "Jetzt Backup erstellen"
  7. Warten Sie, bis das Backup abgeschlossen ist (kann einige Minuten dauern)
Wichtig: Für Backups benötigen Sie eine WLAN-Verbindung und ausreichend iCloud-Speicherplatz. Das Backup läuft automatisch, wenn Ihr Handy geladen wird und mit WLAN verbunden ist.
